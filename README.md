## How I work

I design the prompt and data layer, build the interface, and instrument the analytics so I can tell whether the thing actually worked. I lean on LLMs as a tool, not a crutch: the judgement about what to build and how to verify it stays mine. Most of my projects ship on zero-cost hosting and stay running without me babysitting them.
